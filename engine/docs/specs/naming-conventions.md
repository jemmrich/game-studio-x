# Naming Conventions
This document outlines the naming conventions to be used throughout this repository.

## General Rules
### Files & Folders
- File names: lower-snake-case
- Folder names: lower-snake-case

### Class Names
- Class names: PascalCase
- Use consistent prefixes/suffixes (e.g., RapierRigidBody for library specific modules).

### Variables & Constants
- Variables: camelCase
- Constants: UPPER_SNAKE_CASE

### CSS Class Names
- Use camelCase
